# 时空商旅（Portal Pass）

## 一句话概念
玩家在同一大场景中穿梭于原始时代与科技时代，输入想买的物品，由老板（LLM）按时代给出价值并交易，最终冲击世界首富。

## 核心规则（定版）
- 货币统一为黄金。
- 开局必须先配置 API，并测试成功后才能开始游戏。
- 玩家输入物品名后返回报价与理由。
- 同一物体在两时代价值不同（例：打火机科技时代约 1 金币，古代约 1000 金币）。
- 金币每秒 +1。
- 资产跨阶时弹出提示（例如“当前资产已可购买一辆保时捷”）。

## 胜利目标
- 唯一胜利条件：总资产达到 `targetGold`。
- `targetGold` 由配置文件管理，不写死在脚本中。
